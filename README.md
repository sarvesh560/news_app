# Flutter News App with Firebase Cloud Messaging

A Flutter mobile application that fetches news from a public API and supports push notifications using Firebase Cloud Messaging (FCM).

---

## Features

- Fetches the latest news from [NewsAPI.org](https://newsapi.org/).
- Displays news in a clean, modern UI with:
    - Images
    - Headlines
    - Description
    - Source
- Tap on a news item to view details with a "Read Full Article" button.
- Integrated Firebase Cloud Messaging (FCM) to receive custom push notifications.
- Pull-to-refresh for news updates.
